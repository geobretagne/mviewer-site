Migration des dépôts github

Comme annoncé lors des rencontres mviewer, les dépôts github de mviewer et mviewerstudio ont déménagé en ce début d'année.

Vous retrouverez ces deux projets ici : https://github.com/orgs/mviewer/repositories

Pensez à mettre à jour vos forks et clones des deux projets.

Exemples pour mviewer :
Avant toutes modifications, vérifiez ce qu'il vous faut changer avec la commande:
git remote -v

Clone https:
git remote set-url origin https://github.com/mviewer/mviewer.git

clone ssh:
git remote set-url origin git@github.com:mviewer/mviewer.git

Fork https:
git remote set-url upstream https://github.com/mviewer/mviewer

Fork ssh:
git remote set-url upstream git@github.com:mviewer/mviewer.git

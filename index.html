Propulser vos données

mviewer est une application cartographique initiée et développée par la Région Bretagne, sous le nom de Kartenn (carte en breton).

Le code source de cette application est librement réutilisable depuis 2014. De nombreuses collectivités mais également des entreprises, le secteur de la recherche et de l’enseignement utilisent librement cette application et participent à son évolution.

Découvrir mviewer !

Studio, le générateur d’applications

mviewer studio permet de déployer facilement des applications pour des non spécialistes.
